# Task Manager

This this a basic task manager project.

## Features
1. Add tasks
2. Delete tasks
3. Update Existing tasks
4. Assign tasks as urgent
5. Assign different categories
6. Manage and create new categories


## Technologies used
1. Python
2. FLask
3. MongoDB (mLab)
4. Cloud9 IDE
5. Materialze Framework 

## Deployment 
Deployed on Heroku.
